# Phần mềm nhập điểm

Phần mềm này giúp hỗ trợ quá trình nhập điểm của quý thầy cô được nhanh hơn nhập trực tiếp trên file excel truyền thống.

Thầy cô có thể thêm, sửa, xóa cột điểm và hệ số của cột điểm đó ngay trong phần mềm.

Sau khi nhập thì điểm số sẽ được lưu trực tiếp trên file excel của thầy cô (có 5 phiên bản backup mới nhất)
